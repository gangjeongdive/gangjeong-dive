강정다이브 멀티페이지 홈페이지

변경 내용
- 홈(index.html)은 센터 로고만 보이는 화면입니다.
- 메뉴를 클릭하면 각각 다른 페이지로 이동합니다.

페이지 구성
index.html - 홈 로고 화면
price.html - 가격안내
fundive.html - 펀다이빙
course.html - 교육과정
facility.html - 시설소개
contact.html - 예약문의
location.html - 오시는길

사용 방법
1. 압축을 풀어주세요.
2. VS Code에서 폴더 열기
3. index.html을 Live Server로 실행

이미지 교체는 images 폴더에서 같은 파일명으로 바꾸면 됩니다.


추가된 메뉴
- fundive.html : 펀다이빙 갤러리 페이지
- point.html : 포인트소개 페이지
- images/point-map.svg : 임시 포인트 이미지입니다. 실제 포인트 이미지가 준비되면 같은 이름으로 교체하거나 point.html의 img src를 수정하세요.
